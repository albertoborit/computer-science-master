# Complexity

| Algoritmo        | Complejidad                                        | Memoria      | Método      |
|------------------|----------------------------------------------------|--------------|-------------|
| Bubblesort       | O ( n ²)                                           | O (1)        | Intercambio |
| Merge sort       | O ( n   log   n )                                  | O ( n )      | Mezcla      |
| Quicksort        | Promedio:   O ( n   log   n ),  peor caso: O( n ²) | O (log   n ) | Partición   |
| Binary tree sort | O ( n   log   n )                                  | O( n )       | Inserción   |
